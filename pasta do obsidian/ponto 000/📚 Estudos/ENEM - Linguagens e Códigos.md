## Literatura Brasileira

### Períodos Literários
| Período | Época | Características |
|---------|-------|-----------------|
| Quinhentismo | 1500-1600 | Informativa, catequese |
| Barroco | 1600-1700 | Dualidade, rebuscamento |
| Arcadismo | 1700-1800 | Simplicidade, bucolismo |
| Romantismo | 1800-1880 | Emoção, nacionalismo |
| Realismo/Naturalismo | 1880-1900 | Crítica social, determinismo |
| Parnasianismo | 1880-1900 | Arte pela arte, forma |
| Simbolismo | 1890-1900 | Subjetividade, musicalidade |
| Pré-Modernismo | 1900-1922 | Transição, regionalismo |
| Modernismo | 1922+ | Ruptura, liberdade formal |

### Autores Essenciais
- [ ] Machado de Assis — Realismo, ironia
