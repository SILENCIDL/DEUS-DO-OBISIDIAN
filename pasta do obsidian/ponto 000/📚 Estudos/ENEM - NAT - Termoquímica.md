# 📗 Termoquímica

Categoria: [[ENEM - Ciências da Natureza]]

---

## O que estuda?

Estudo das **trocas de calor** (energia) nas reações químicas.

---

## Conceitos Básicos

| Termo | Significado |
|-------|-------------|
| **Entalpia (H)** | Conteúdo energético de uma substância |
| **ΔH** | Variação de entalpia (Hfinal - Hinicial) |
| **Exotérmica** | Libera calor (ΔH < 0) |
| **Endotérmica** | Absorve calor (ΔH > 0) |

---

## Reações Exotérmicas e Endotérmicas

### Exotérmica (ΔH < 0)
- **Libera** energia para o ambiente
- Produtos têm **menos** energia que reagentes
- Exemplo: Combustão

```
CH₄ + 2O₂ → CO₂ + 2H₂O   ΔH = -890 kJ
```

### Endotérmica (ΔH > 0)
- **Absorve** energia do ambiente
- Produtos têm **mais** energia que reagentes
- Exemplo: Fotossíntese

```
6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂   ΔH = +2800 kJ
```

---

## Tipos de Entalpia

### Entalpia de Formação (ΔHf)
Energia para formar 1 mol da substância a partir dos elementos.
- Substâncias simples têm ΔHf = 0

### Entalpia de Combustão (ΔHc)
Energia liberada na queima completa de 1 mol.
- Sempre negativa (exotérmica)

### Entalpia de Ligação
Energia para **romper** 1 mol de ligações.
- Sempre positiva (precisa de energia)

---

## Lei de Hess

> "A variação de entalpia de uma reação depende apenas dos estados inicial e final, não do caminho."

**Aplicação**: Somar reações intermediárias para obter ΔH da reação total.

### Exemplo
Se:
- A → B: ΔH₁ = -100 kJ
- B → C: ΔH₂ = -50 kJ

Então:
- A → C: ΔH = ΔH₁ + ΔH₂ = -150 kJ

---

## Cálculo de ΔH por Entalpia de Formação

> **ΔH = Σ ΔHf (produtos) - Σ ΔHf (reagentes)**

---

## Cálculo de ΔH por Energia de Ligação

> **ΔH = Σ (ligações rompidas) - Σ (ligações formadas)**

- Romper ligação: absorve energia (+)
- Formar ligação: libera energia (-)

---

## Combustíveis

| Combustível | Entalpia de combustão |
|-------------|----------------------|
| Hidrogênio | Alta (mais energia/massa) |
| Gasolina | Alta |
| Etanol | Média |
| Carvão | Média |
| Madeira | Baixa |

### Biocombustíveis
- **Etanol**: Da cana-de-açúcar
- **Biodiesel**: De óleos vegetais
- **Vantagem**: Renovável, menos CO₂ líquido

---

## Dica ENEM

O ENEM cobra:
1. **Identificar exo/endotérmica** pelo sinal de ΔH
2. **Lei de Hess** — somar reações
3. **Energia de ligação** — cálculo de ΔH
4. **Combustíveis** — eficiência energética

**Atenção ao sinal!**
- ΔH < 0 (negativo) = libera = exotérmica
- ΔH > 0 (positivo) = absorve = endotérmica
