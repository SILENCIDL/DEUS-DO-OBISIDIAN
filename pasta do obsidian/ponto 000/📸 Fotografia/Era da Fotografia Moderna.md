# 💡 Era da Fotografia Moderna (1880-1920)

> *"A fotografia se torna acessível a todos — 'You press the button, we do the rest.'"*

---

## 🎯 Contexto Histórico

A partir de **1880**, a fotografia deixou de ser um processo complexo de laboratório e se tornou uma atividade acessível ao público em geral. A invenção do **filme flexível** revolucionou a forma como as pessoas capturavam imagens.

```

│           ERA DA FOTOGRAFIA MODERNA (1880-1920)             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  🎯 REVOLUÇÃO PRINCIPAL:                                    │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                                                     │   │
│  │   FILME FLEXÍVEL → CÂMERA PORTÁTIL → FOTO PARA TODOS│   │
│  │                                                     │   │
│  │   • Não precisa de laboratório móvel               │   │
│  │   • Câmeras pequenas e leves                       │   │
│  │   • Revelação feita pela fábrica                   │   │
│  │   • Custo acessível                                │   │
│  │                                                     │   │
│  └─────────────────────────────────────────────────────┘   │

## 📅 Linha do Tempo

### A Revolução Kodak

| Ano | Evento | Significado |
|-----|--------|-------------|
| **1880** | George Eastman funda Eastman Dry Plate | Início da indústria |
| **1884** | Filme em rolo patenteado | Flexibilidade |
| **1888** | [[Kodak nº 1]] lançada | "You press the button, we do the rest" |
| **1889** | Filme de celulose transparente | Padrão moderno |
| **1900** | [[Brownie Camera]] | Custa US$ 1 — acessível a todos |
| **1913** | [[Oskar Barnack]] desenvolve proto-Leica | 35mm compacto |
| **1925** | [[Leica I]] comercializada | Revolução do 35mm |

## 🎨 Movimentos desta Era

### [[Pictorialismo]] (1885-1915)

- Fotografia como **arte pictórica**
- Foco suave, manipulação química
- Impressões em papel especial
- [[Alfred Stieglitz]] como líder

### [[Fotojornalismo]] (1900+)

- Registro de eventos em tempo real
- [[Mathew Brady]] — Guerra Civil EUA
- [[Jacob Riis]] — Documentação social
- [[Lewis Hine]] — Trabalho infantil

## 🏆 Legado desta Era

### Conquistas

1. **Democratização**: Fotografia para todos
2. **Indústria**: Kodak se torna gigante global
3. **Memória**: Álbuns de família se popularizam
4. **Jornalismo**: Fotojornalismo nasce
5. **Arte**: Fotografia reconhecida como arte

### Transição para a Próxima Era

```
