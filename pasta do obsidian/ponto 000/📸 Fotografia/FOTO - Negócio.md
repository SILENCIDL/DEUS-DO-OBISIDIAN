# FOTO — Negócio
## Módulo 3: Transformar Fotografia em Renda Real

> "Talento sem estratégia de negócio é hobby. Negócio sem talento é fraude. Junte os dois."

---

# AULA 1 — Mentalidade de Fotógrafo Empreendedor

## A mudança de chave

| Mentalidade de hobbysta | Mentalidade de profissional |
|-------------------------|----------------------------|
| "Fotografo porque gosto" | "Fotografo porque gosto E porque gera valor" |
| Cobra barato por insegurança | Cobra justo pelo valor que entrega |
| Aceita qualquer trabalho | Escolhe trabalhos estratégicos |
| Não tem processo | Tem workflow completo |
| Posta quando dá vontade | Publica com estratégia |
| Espera clientes virem | Vai atrás e constrói demanda |

## Os 4 pilares do fotógrafo-negócio

1. **Produto** — O que você vende (não é "foto", é transformação)
2. **Público** — Para quem você vende (nicho específico)
3. **Posicionamento** — Como você é percebido (marca pessoal)
4. **Processo** — Como você entrega (do contato à entrega final)

# AULA 2 — Definindo seu Nicho

## Por que nichar

- Fotógrafo genérico compete com todo mundo por preço
- Fotógrafo de nicho compete por **especialidade e confiança**
- Nicho permite cobrar mais, atrair melhor, se posicionar como referência

# AULA 3 — Precificação (Quanto Cobrar)

## A fórmula base

```
PREÇO DO SERVIÇO = Custos + Tempo + Lucro + Valor percebido
```

### Passo 1: Calcule seus custos fixos mensais

| Custo | Valor mensal |
|-------|-------------|
| Aluguel/Moradia | R$ _____ |
| Internet | R$ _____ |
| Luz/Água | R$ _____ |
| Alimentação | R$ _____ |
| Transporte | R$ _____ |
| MEI/Imposto | R$ _____ |
| Equipamento (depreciação) | R$ _____ |
| Software (Adobe, etc.) | R$ _____ |
| Marketing/Ads | R$ _____ |
| Reserva de emergência (10%) | R$ _____ |
| **TOTAL CUSTOS** | **R$ _____** |

### Passo 2: Defina quantos trabalhos faz por mês

- Dias úteis trabalhando: _____ dias
- Trabalhos por semana (realista): _____
- **Trabalhos por mês**: _____

### Passo 3: Calcule o mínimo por trabalho

```
MÍNIMO POR TRABALHO = Custos mensais ÷ Trabalhos por mês
```

### Passo 4: Adicione lucro (30-50% em cima)

```
PREÇO BASE = Mínimo × 1.3 (30% lucro) a 1.5 (50% lucro)
```

### Passo 5: Ajuste pelo valor percebido

| Fator | Aumenta preço | Diminui preço |
|-------|---------------|---------------|
| Experiência | Muitos anos, portfólio forte | Iniciante |
| Demanda | Lista de espera | Pouca procura |
| Localização | Capital, região rica | Interior, região simples |
| Nicho | Especializado | Genérico |
| Entregáveis | Muitos (álbum, impressão, vídeo) | Só digitais |
| Urgência | Prazo curto | Prazo normal |

### Fotógrafo intermediário (1-3 anos)

| Serviço | Preço mínimo | Preço ideal |
|---------|-------------|-------------|
| Ensaio pessoal (1-2h, 30 fotos) | R$500 | R$900 |
| Ensaio família (2h, 40 fotos) | R$600 | R$1.200 |

### Erros de precificação

| Erro | Consequência | Solução |
|------|-------------|---------|
| Cobrar muito barato | Desvaloriza o mercado todo | Calcular custos reais |
| Cobrar sem contrato | Cliente pede infinitas edições | Sempre ter contrato |
| Não cobrar deslocamento | Prejuízo em trabalhos longe | Incluir no orçamento |
| Dar todas as fotos | Sem controle de uso | Definir quantidade no contrato |
| Comparar com amador | Preço de hobby, não profissão | Comparar com profissionais |

# AULA 4 — Captação de Clientes

## Os 5 canais de captação

### 1. Instagram (principal para fotógrafos)
- Portfólio visual constante
- Stories com bastidores
- Reels com antes/depois
- Destaques organizados por tipo de trabalho
- Link na bio para WhatsApp/portfólio
- Hashtags do nicho e da região

### 2. Indicação (boca a boca)
- Melhor tipo de cliente (já vem confiando)
- Pedir ativamente para clientes indicarem
- Oferecer desconto para quem indica
- Fazer um trabalho tão bom que indiquem naturalmente

### 3. Google Meu Negócio
- Cadastrar como fotógrafo na sua cidade
- Pedir reviews de clientes
- Subir fotos do seu trabalho
- Aparece quando alguém busca "fotógrafo em [cidade]"

### 4. Parcerias locais
| Parceiro | O que oferecer | O que receber |
|----------|---------------|---------------|
| Salões de beleza | Foto para divulgação | Indicação de noivas/clientes |
| Buffets/Eventos | Fotos dos eventos | Indicação de clientes |
| Lojas de roupa | Fotos de produto | Exposição e indicação |
| Decoradores | Fotos de ambientes | Clientes de imóveis |
| Maquiadores | Fotos do trabalho | Indicação mútua |

### 5. Presencial/Networking
- Participar de feiras e eventos
- Levar cartão de visita
- Conversar, não vender
- Oferecer uma sessão teste para primeiro contato

# AULA 5 — Atendimento e Experiência do Cliente

## O workflow profissional completo

### Fase 1: Primeiro contato
- [ ] Responder em até 2 horas
- [ ] Perguntar: O que precisa? Quando? Onde? Quantas fotos?
- [ ] Enviar portfólio relevante (não tudo, só o que importa para ele)
- [ ] Enviar proposta com preço + o que inclui

### Fase 2: Fechamento
- [ ] Enviar contrato (SEMPRE)
- [ ] Cobrar sinal (30-50% antecipado)
- [ ] Alinhar expectativas por escrito
- [ ] Confirmar data, hora, local
- [ ] Enviar guia de preparação para o cliente

### Fase 3: Produção
- [ ] Chegar antes do horário combinado
- [ ] Verificar equipamento antes
- [ ] Ser profissional e pontual
- [ ] Dirigir a sessão com confiança
- [ ] Fazer mais fotos do que o contratado (margem de seleção)

### Fase 4: Pós-produção
- [ ] Selecionar melhores fotos
- [ ] Editar com consistência
- [ ] Entregar no prazo combinado
- [ ] Enviar preview antes da galeria completa
- [ ] Entregar via galeria online (Google Drive, Pic-Time, etc.)

### Fase 5: Pós-entrega
- [ ] Pedir feedback
- [ ] Pedir review no Google
- [ ] Pedir permissão para postar no portfólio
- [ ] Enviar mensagem de agradecimento
- [ ] Oferecer próximo serviço (ensaio sazonal, por exemplo)

# AULA 6 — Portfólio e Marca Pessoal

## Construindo um portfólio que vende

### Regras do portfólio

| Regra | Por quê |
|-------|---------|
| Menos é mais | 20 fotos incríveis > 200 medianas |
| Consistência visual | Mostra que você tem estilo |
| Só mostre o que quer fazer mais | Vai atrair trabalhos semelhantes |
| Atualize a cada 3 meses | Portfólio velho = profissional parado |
| Organize por categoria | Facilita a navegação do cliente |

### Onde ter portfólio

| Plataforma | Preço | Profissionalismo | Para quem |
|------------|-------|-------------------|-----------|
| Instagram | Grátis | Médio | Todo fotógrafo |
| Behance | Grátis | Alto | Criativos |
| Flickr/500px | Grátis | Médio | Comunidade foto |
| Site próprio (WordPress) | R$15-50/mês | Muito alto | Profissional sério |
| Squarespace | R$60-80/mês | Muito alto | Portfólio premium |
| Notion (público) | Grátis | Médio | Temporário/rápido |

### Estrutura ideal do portfólio
1. **Página inicial**: 5-10 melhores fotos (impacto visual)
2. **Sobre mim**: Quem você é, como trabalha, sua visão
3. **Galerias por categoria**: Retrato, Evento, Produto, etc.
4. **Depoimentos**: 3-5 reviews de clientes
5. **Contato**: WhatsApp, email, formulário
6. **Preços (opcional)**: "A partir de R$..." ou "Solicite orçamento"

# AULA 7 — Escalando o Negócio

## Fases de crescimento

### Fase 1: Sobrevivência (0-6 meses)
- Aceitar trabalhos variados para aprender
- Construir portfólio
- Cobrar preço justo (não de graça)
- Meta: pagar as contas com fotografia

### Fase 2: Estabilidade (6-18 meses)
- Definir nicho principal
- Ter fluxo constante de clientes
- Subir preços gradualmente
- Meta: renda previsível mensal

### Fase 3: Crescimento (18-36 meses)
- Múltiplas fontes de renda (serviço + produto)
- Produtos digitais (presets, curso, e-book)
- Presença forte online
- Meta: renda passiva + ativa

### Fase 4: Escala (36+ meses)
- Contratar assistente/segundo fotógrafo
- Terceirizar edição
- Criar marca consolidada
- Meta: negócio que funciona sem você 100% do tempo

## MEI para fotógrafo

### Como regularizar
1. Acessar gov.br/mei
2. Cadastrar como "Fotógrafo(a) independente" (CNAE 7420-0/01)
3. Pagar DAS mensal (~R$75)
4. Emitir nota fiscal pelo app MEI

### Limites do MEI
- Faturamento máximo: R$81.000/ano (R$6.750/mês)
- Sem empregado (ou apenas 1)
- Se ultrapassar: migrar para ME (Microempresa)

### Vantagens
- CNPJ (passa mais confiança)
- Nota fiscal (exigido por empresas)
- INSS (aposentadoria)
- Baixo custo
