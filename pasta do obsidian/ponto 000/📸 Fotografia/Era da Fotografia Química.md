# 🎞️ Era da Fotografia Química (1839-1880)

> *"O período em que a fotografia nasceu, cresceu e se estabeleceu como realidade comercial."*

---

## 🎯 Contexto Histórico

Em **1839**, a fotografia foi oficialmente "presenteada ao mundo". Este período é marcado por intensa experimentação química, com diferentes processos competindo pelo domínio do mercado.

## 📸 [[Daguerreótipo]] (1839-1860)

### O Primeiro Processo Comercial

```
┌─────────────────────────────────────────────┐
│           DAGUERRÉOTIPO                     │
│                                             │
│  📅 1839 — Paris, França                    │
│  👤 Louis Daguerre                          │
│                                             │
│  🔬 PROCESSO:                               │
│  1. Placa de cobre banhada a prata         │
│  2. Polimento até espelho                  │
│  3. Sensibilização com iodo (vapor)        │
│  4. Exposição na câmera (30 min → 1 min)   │
│  5. Revelação com vapor de mercúrio        │
│  6. Fixação com hipossulfito de sódio      │
│                                             │
│  🖼️ RESULTADO:                              │
│  - Imagem única (não copiável)             │
│  - Espelhada (positivo direto)             │
│  - Detalhes incríveis                      │
│  - Superfície frágil                       │
│                                             │
└─────────────────────────────────────────────┘
```

### Características

| Aspecto | Descrição |
|---------|-----------|
| **Tempo** | Inicialmente 30 min, depois 1-2 min |
| **Imagem** | Positivo direto em metal |
| **Cópias** | Não era possível (imagem única) |
| **Custo** | Alto (acessível apenas para ricos) |
| **Qualidade** | Extremamente nítida |

### Curiosidade
> O daguerreótipo era tão detalhado que se dizia ser possível contar os cabelos de uma pessoa em um retrato.

## 📄 [[Calótipo de Fox Talbot]] (1841-1860)

### O Processo de Negativo

```
┌─────────────────────────────────────────────┐
│           CALÓTIPO (TALBOTIPO)              │
│                                             │
│  📅 1841 — Inglaterra                       │
│  👤 William Henry Fox Talbot                │
│                                             │
│  🔬 PROCESSO:                               │
│  1. Papel de boa qualidade                  │
│  2. Banho em nitrato de prata              │
│  3. Secagem                                │
│  4. Banho em ácido gálico (sensibilizante) │
│  5. Exposição na câmera (1-5 min)          │
│  6. Revelação com ácido gálico quente      │
│  7. Fixação em hipossulfito                │
│                                             │
│  🖼️ RESULTADO:                              │
│  - NEGATIVO em papel                        │
│  - Permite CÓPIAS positivas                │
│  - Imagem menos nítida (fibra do papel)    │
│                                             │
└─────────────────────────────────────────────┘
```

### Vantagens sobre o Daguerreótipo

| Calótipo | Daguerreótipo |
|----------|---------------|
| ✅ Cópias ilimitadas | ❌ Imagem única |
| ✅ Papel (mais barato) | ❌ Metal caro |
| ✅ Processo mais simples | ❌ Mercúrio tóxico |
| ❌ Menos nítido | ✅ Extremamente nítido |

### A Guerra dos Processos

```
┌─────────────────────────────────────────────┐
│                                             │
│   FRANÇA 🇫🇷          vs.        INGLATERRA 🇬🇧│
│                                             │
│   Daguerreótipo                Calótipo    │
│   ─────────────                ─────────   │
│   • Imagem única               • Negativo  │
│   • Nítida                     • Cópias    │
│   • Popular inicialmente       • Futuro    │
│                                             │
│   VENCEDOR A LONGO PRAZO: Calótipo         │
│   (por permitir cópias)                     │
│                                             │
└─────────────────────────────────────────────┘
```

## 🧪 [[Colódio Úmido de Archer]] (1851-1880)

### A Revolução do Vidro

```
┌─────────────────────────────────────────────┐
│           COLÓDIO ÚMIDO                     │
│                                             │
│  📅 1851 — Inglaterra                       │
│  👤 Frederick Scott Archer                  │
│                                             │
│  🔬 PROCESSO:                               │
│  1. Placa de vidro limpa                   │
│  2. Banho em colódio (nitrato de celulose) │
│  3. Sensibilização em nitrato de prata     │
│  4. Exposição (5-30 segundos!)             │
│  5. Revelação imediata (úmido)             │
│  6. Fixação e lavagem                      │
│                                             │
│  ⚠️ PROBLEMA: A placa precisava estar      │
│     ÚMIDA durante todo o processo!         │n│     → Laboratório móvel obrigatório       │
│                                             │
│  🖼️ RESULTADO:                              │
│  - NEGATIVO em vidro                        │
│  - Extremamente NÍTIDO                      │
│  - Permite CÓPIAS                           │
│  - Tempo de exposição CURTO                 │
│                                             │
└─────────────────────────────────────────────┘
```

### O Laboratório Móvel

Fotógrafos precisavam levar uma **tenda escura** para revelar as placas imediatamente:

```
┌─────────────────────────────────────────────┐
│                                             │
│        🏕️ TENDA ESCURA (DARKROOM)          │
│        ┌─────────────────────┐              │
│        │  🧪 Produtos        │              │
│        │  💧 Água destilada  │              │
│        │  🥃 Colódio         │              │
│        │  🧂 Nitrato de prata│              │
│        │  🧴 Revelador       │              │
│        │  🧼 Fixador         │              │
│        └─────────────────────┘              │
│                                             │
│  → O fotógrafo tinha que revelar            │
│    ANTES da placa secar!                    │
│                                             │
└─────────────────────────────────────────────┘
```

### Impacto

| Área | Impacto |
|------|---------|
| **Retratos** | Tornou-se mais acessível (tempo menor) |
| **Paisagens** | Possível fotografar ao ar livre |
| **Guerra** | Documentação de campo |
| **Arquitetura** | Registro de construções |

## 🎨 Aplicações Importantes

### [[Fotografia de Guerra]] — Crimeia (1855)

**Roger Fenton** documentou a Guerra da Crimeia:
- Primeiro registro fotográfico de conflito armado
- Imagens de campo de batalha
- Limitação: não mostrava mortos (censura)

### [[Retratos Carte de Visite]] (1860s)

- **André-Adolphe-Eugène Disdéri** patentou o formato
- 8 poses em uma única placa (econômico)
- Democratizou o retrato fotográfico
- Celebridades vendiam suas fotos

### Fotografia de Paisagem

- **Carleton Watkins** — Yosemite (EUA)
- **Francis Frith** — Egito e Oriente Médio
- Influenciaram a arte e o turismo

## 🏆 Legado desta Era

### Conquistas

1. **Fotografia como realidade**: Deixou de ser curiosidade científica
2. **Comercialização**: Estúdios fotográficos proliferaram
3. **Documentação**: Registro histórico sistemático
4. **Arte**: Reconhecimento como forma de expressão
