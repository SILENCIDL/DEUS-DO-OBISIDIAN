# 📱 Era da Fotografia Digital (1990-Atual)

> *"A revolução dos pixels — da câmera profissional ao smartphone no bolso."*

---

## 🎯 Contexto Histórico

A partir de **1990**, a fotografia passou por sua maior transformação desde a invenção do filme flexível. A substituição da química pela eletrônica revolucionou como criamos, armazenamos e compartilhamos imagens.

## 🏆 Legado desta Era

### Conquistas

1. **Acesso universal**: Qualquer pessoa com smartphone fotografa
2. **Instantaneidade**: Ver a foto imediatamente
3. **Custo zero**: Sem filme, sem revelação
4. **Compartilhamento**: Global e imediato
5. **Edição**: Pós-processamento acessível
6. **Armazenamento**: Milhares de fotos em um cartão

### Desafios

```
┌─────────────────────────────────────────────┐
│         DESAFIOS DA ERA DIGITAL             │
├─────────────────────────────────────────────┤
│                                             │
│  📸 SOBRECARGA:                             │
│  → Milhares de fotos, nenhuma impressa     │
│  → Dificuldade de organizar                │
│  → "Paradoxo da escolha"                   │
│                                             │
│  🎨 VALOR:                                  │
│  → Fotos descartáveis                      │
│  → Menos atenção a cada imagem             │
│  → Cultura do "descartável"                │
│                                             │
│  🔒 PRIVACIDADE:                            │
│  → Fotos na nuvem                          │
│  → Risco de vazamentos                     │
│  → Uso comercial de imagens                │
│                                             │
│  🧠 MEMÓRIA:                                │
│  → Dependência do digital                  │
│  → Perda de fotos por falha técnica        │
│  → Álbuns físicos em extinção              │
│                                             │
└─────────────────────────────────────────────┘
```
