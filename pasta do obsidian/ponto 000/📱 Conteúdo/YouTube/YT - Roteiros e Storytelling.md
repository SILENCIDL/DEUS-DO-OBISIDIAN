# YT — Roteiros e Storytelling

> Um bom roteiro transforma um vídeo mediano em algo que as pessoas não conseguem parar de assistir.

---

## Estruturas de roteiro

### 1. Estrutura HOOK (Para tutoriais)
```
GANCHO (0-15s): Mostre o resultado ou faça uma pergunta
CONTEXTO (15-45s): Por que isso importa
CONTEÚDO (45s - X min): Passo a passo
RECAP (últimos 60s): Resumo rápido
CTA (últimos 15s): Inscreva-se, comente, próximo vídeo
```

### 2. Estrutura PROBLEMA-SOLUÇÃO (Para dicas)
```
PROBLEMA: "Você já tentou fotografar e ficou tudo escuro?"
AGITAÇÃO: "Isso acontece porque..."
SOLUÇÃO: "A técnica que resolve é..."
PROVA: Mostrar resultado
CTA: Próximo passo
```

### 3. Estrutura STORYTELLING (Para vlogs/bastidores)
```
ABERTURA: Momento impactante do meio/final
CONTEXTO: Como chegou ali
JORNADA: Desenvolvimento da história
CLÍMAX: O momento principal
REFLEXÃO: O que aprendeu
```

### 4. Estrutura LISTA (Para rankings/compilações)
```
GANCHO: "Vou mostrar os X melhores..."
INTRODUÇÃO: Critérios de seleção
ITENS: Do menos ao mais importante (mantém curiosidade)
BÔNUS: Item extra inesperado
CTA: "Qual seu favorito? Comenta!"
```

## Técnicas de storytelling

### Os 5 elementos de uma boa história
1. **Personagem**: Você, o espectador, um cliente
2. **Conflito**: O problema que precisa resolver
3. **Jornada**: O caminho para a solução
4. **Transformação**: O antes e depois
5. **Lição**: O que fica para o espectador

### Ganchos emocionais
| Emoção | Gatilho | Exemplo |
|--------|---------|---------|
| Curiosidade | Informação incompleta | "Existe um truque que..." |
| Medo | Perda/Erro | "Você está destruindo suas fotos sem saber" |
| Desejo | Resultado aspiracional | "Como tirar fotos como profissional" |
| Surpresa | Revelação inesperada | "A verdade sobre lentes baratas" |
| Pertencimento | Comunidade | "Todo fotógrafo passa por isso" |
| Urgência | Tempo limitado | "Faça isso ANTES de perder oportunidades" |

## Transições de roteiro (para manter fluidez)

- "Mas antes de continuar..."
- "E aqui é onde fica interessante..."
- "Agora, o mais importante..."
- "Mas tem um detalhe que muda tudo..."
- "E foi aí que eu descobri..."
- "O que ninguém fala é..."
- "Presta atenção nisso aqui..."

---

## Erros comuns em roteiros

| Erro | Problema | Solução |
|------|----------|---------|
| Intro longa | Perde espectador | Gancho em 5 segundos |
| Sem estrutura | Confuso, desengaja | Seguir template |
| Falar demais | Monótono | Mais B-roll, cortes rápidos |
| Sem CTA | Não converte | Sempre pedir ação |
| Roteiro decorado | Parece robótico | Use bullet points, não texto completo |
| Sem loop aberto | Saem antes do final | "Daqui a pouco mostro..." |
