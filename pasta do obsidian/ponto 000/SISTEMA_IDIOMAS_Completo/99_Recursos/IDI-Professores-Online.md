## 📋 Plataformas Comparadas

### italki

| Aspecto | Detalhe |
|---------|---------|
| **Preço** | $5-40/aula |
| **Professores** | 10.000+ |
| **Trial** | Aula de teste (preço reduzido) |
| **Créditos** | Compre créditos, use quando quiser |
| **Melhor para** | Variedade de professores |

**Como funciona:**
1. Crie conta
2. Procure professores (filtre por idioma, preço, país)
3. Agende aula de trial
4. Se gostar, compre mais aulas

**Dica:** Procure professores com +100 aulas e avaliação 5 estrelas.

---

### Preply

| Aspecto | Detalhe |
|---------|---------|
| **Preço** | $5-40/aula |
| **Professores** | 10.000+ |
| **Trial** | Aula de teste (preço reduzido) |
| **Assinatura** | Pacotes mensais com desconto |
| **Melhor para** | Agendamento fácil |

**Como funciona:**
1. Crie conta
2. Faça teste de nível (opcional)
3. Receba recomendações de professores
4. Agende aula de trial

**Dica:** Use o filtro "Super Tutor" para os melhores.

## 🌍 Sotaques: Espanha vs América Latina

### Professores da Espanha

| Vantagens | Desvantagens |
|-----------|--------------|
| Sotaque "padrão" | Ceceo (th em vez de s) |
| Vocabulário europeu | Pode ser difícil no início |
| Útil para intercâmbio na Espanha | | |

**Bom para:** Intercâmbio na Espanha, exames DELE.

### Professores da América Latina

| País | Característica do Sotaque |
|------|---------------------------|
| **México** | Neutro, claro |
| **Colômbia** | Considerado o mais "padrão" |
| **Argentina** | Voseo ("vos" em vez de "tú") |
| **Cuba** | Sotaque caribenho, "s" mudo |
| **Peru** | Claro, devagar |

**Bom para:** Conversação geral, variedade cultural.

## 💰 Economizando em Aulas

### Dicas

1. **Aulas em grupo** — Mais barato (italki oferece)
2. **Pacotes** — Desconto por comprar várias
3. **Professores iniciantes** — Preço menor, qualidade boa
4. **Tandem** — Troca de idiomas (grátis!)

### Orçamento Mensal Sugerido

| Nível | Aulas/semana | Custo/aula | Total/mês |
|-------|--------------|------------|-----------|
| Iniciante | 1 | $10 | ~R$200 |
| Intermediário | 2 | $12 | ~R$480 |
| Avançado | 2 | $15 | ~R$600 |

## 📝 Preparando para a Aula

### Antes da Aula
- [ ] Definir objetivo da aula
- [ ] Preparar perguntas
- [ ] Revisar conteúdo anterior
- [ ] Testar equipamento (câmera, microfone)

### Durante a Aula
- [ ] Falar o máximo possível
- [ ] Pedir correções
- [ ] Anotar erros e novas palavras
- [ ] Perguntar quando não entender

### Depois da Aula
- [ ] Revisar anotações
- [ ] Praticar o que aprendeu
- [ ] Agendar próxima aula
