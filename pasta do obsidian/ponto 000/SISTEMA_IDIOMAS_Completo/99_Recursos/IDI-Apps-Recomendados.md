## 📚 Apps por Categoria

### Flashcards e Memorização

| App | Preço | Melhor para |
|-----|-------|-------------|
| **Anki** | Grátis (Android) / $25 (iOS) | Sistema SRS, totalmente personalizável |
| **Quizlet** | Freemium | Jogos e modos de estudo variados |
| **Memrise** | Freemium | Vídeos de nativos |
| **Tinycards** | Grátis | Interface bonita, simples |

**Recomendação:** Anki para sério, Quizlet para iniciantes.

## 🎯 Configuração Ideal para Espanhol

### Iniciante (A1-A2)
1. **Duolingo** — 15 min/dia
2. **Anki** — 50 flashcards/dia
3. **SpanishDict** — consultas
4. **YouTube** — Dreaming Spanish

### Intermediário (B1-B2)
1. **italki** — 2 aulas/semana
2. **Anki** — 100 flashcards/dia
3. **Netflix** — séries com legendas ES
4. **Tandem** — chat diário

### Avançado (C1+)
1. **italki** — aulas de conversação
2. **YouTube** — conteúdo nativo
3. **Podcasts** — sem legenda
4. **Livros** — literatura original
