## Modelos de Contratação

| Modelo | Vantagens | Desvantagens |
|--------|-----------|--------------|
| CLT | Dedicação, vínculo | Custo alto, burocracia |
| PJ | Flexibilidade | Menor vínculo |
| Freelancer | Sob demanda | Disponibilidade |
| Parceria | Sem custo fixo | Divisão de lucro |

### Custos reais de cada modelo:

**CLT:**
```
Salário: R$3.000
+ FGTS (8%): R$240
+ INSS patronal (20%): R$600
+ Férias + 1/3: R$333
+ 13º: R$250
+ Provisões: R$200
= CUSTO REAL: ~R$4.600/mês
```

**PJ:**
```
Pagamento: R$4.000
= CUSTO REAL: R$4.000/mês
(Sem encargos, profissional paga próprios impostos)
```

**Freelancer:**
```
Por projeto: R$500-2.000
Sob demanda, sem compromisso mensal
```

## Delegação Efetiva

### Matriz de delegação:

```
                    IMPORTÂNCIA
                   ALTA │ BAIXA
              ┌─────────┼─────────┐
         ALTA │  FAZER  │ DELEGAR │
    URGÊNCIA  │  VOCÊ   │ (MONIT.)│
              ├─────────┼─────────┤
        BAIXA │ AGENDAR │ DELEGAR │
              │         │(AUTÔNOM)│
              └─────────┴─────────┘
```

### O que delegar primeiro:

1. **Tarefas repetitivas** (agendamento, respostas)
2. **Tarefas técnicas** (edição, design)
3. **Tarefas operacionais** (financeiro, relatórios)
4. **Tarefas de suporte** (atendimento, SAC)

### O que NÃO delegar (no início):

- ❌ Estratégia de conteúdo
- ❌ Relacionamento com audiência
- ❌ Decisões de negócio
- ❌ Gravação (você é a marca)
- ❌ Vendas high-ticket

## Funções Detalhadas

### Assistente Virtual

**Responsabilidades:**
- Gerenciar agenda
- Responder emails/DMs
- Organizar arquivos
- Pesquisas
- Tarefas administrativas

**Ferramentas que deve dominar:**
- Google Workspace
- Notion/Trello
- Canva básico
- Redes sociais

### Editor de Vídeo

**Responsabilidades:**
- Editar vídeos longos (YouTube)
- Editar Reels/Shorts
- Cortes de podcast
- Legendas
- Thumbnails

**Ferramentas que deve dominar:**
- Premiere Pro ou DaVinci
- After Effects (diferencial)
- CapCut
- Photoshop

### Social Media

**Responsabilidades:**
- Calendário de conteúdo
- Agendamento de posts
- Interação com audiência
- Relatórios de métricas
- Pesquisa de trends

**Ferramentas que deve dominar:**
- Meta Business Suite
- mLabs/Later
- Canva
- Planilhas

### Gestor de Tráfego

**Responsabilidades:**
- Criar campanhas
- Otimizar anúncios
- Gerenciar orçamento
- Análise de resultados
- Testes A/B

**Ferramentas que deve dominar:**
- Meta Ads
- Google Ads
- Google Analytics
- Planilhas avançadas

## Gestão de Equipe

### Reuniões:

| Tipo | Frequência | Duração | Objetivo |
|------|------------|---------|----------|
| Daily | Diária | 15min | Alinhamento |
| Weekly | Semanal | 1h | Planejamento |
| 1:1 | Quinzenal | 30min | Feedback |
| Mensal | Mensal | 2h | Estratégia |

### Ferramentas de gestão:

| Ferramenta | Uso |
|------------|-----|
| Slack/Discord | Comunicação |
| Notion | Documentação |
| Trello/Asana | Tarefas |
| Google Meet | Reuniões |
| Loom | Vídeos assíncronos |

### Boas práticas:

1. **Documentar tudo** (SOPs)
2. **Comunicação clara** (expectativas)
3. **Feedback constante** (não acumular)
4. **Autonomia gradual** (ganhar confiança)
5. **Metas claras** (mensuráveis)

## Processo de Contratação

### 1. Definir a vaga
- Responsabilidades claras
- Requisitos técnicos
- Soft skills desejadas
- Faixa salarial

### 2. Divulgar
- LinkedIn
- Plataformas de freelancer
- Stories/Comunidade
- Indicações

### 3. Filtrar candidatos
- Currículo/Portfolio
- Teste prático
- Entrevista inicial

### 4. Entrevista final
- Fit cultural
- Expectativas alinhadas
- Negociação

### 5. Onboarding
- Apresentar a empresa
- Treinar ferramentas
- Definir primeiras tarefas
- Acompanhar de perto
