## 🎯 Visão Geral

Este calendário foi criado para você **planejar, organizar e executar** sua estratégia de conteúdo de forma consistente. A consistência é o segredo dos criadores de sucesso.

# 📝 TEMPLATES DE LEGENDA

## Template 1: AIDA
```
🔥 [ATENÇÃO - Gancho forte]

[INTERESSE - Desenvolva o problema/curiosidade]

[DESEJO - Mostre a solução/benefício]

👇 [AÇÃO - CTA claro]

#hashtag1 #hashtag2 #hashtag3
```

## Template 2: Storytelling
```
[Situação inicial - contexto]

Mas então...

[Conflito/problema]

Foi quando eu descobri que...

[Solução/aprendizado]

💡 [Lição/insight para o seguidor]

Comenta aqui se você já passou por isso! 👇
```

## Template 3: Lista de Valor
```
📌 [X] [TÍTULO DO TEMA] que você precisa saber:

1️⃣ [Item 1]
2️⃣ [Item 2]
3️⃣ [Item 3]
4️⃣ [Item 4]
5️⃣ [Item 5]

Qual desses você já sabia? 🤔

Salva esse post pra consultar depois! 💾
```

## Template 4: Pergunta Engajadora
```
Me conta uma coisa...

[Pergunta relacionada ao nicho]

Eu começo: [Sua resposta]

Agora é sua vez! 👇
```

## Template 5: Oferta
```
🚨 [URGÊNCIA/NOVIDADE]

Se você quer [RESULTADO DESEJADO]...

Isso é pra você 👇

✅ [Benefício 1]
✅ [Benefício 2]
✅ [Benefício 3]

🎁 [BÔNUS/OFERTA ESPECIAL]

⏰ [ESCASSEZ - Vagas limitadas/tempo limitado]

🔗 Link na bio | Comenta "QUERO" que te mando!
```
