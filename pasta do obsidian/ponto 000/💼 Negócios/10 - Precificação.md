## Métodos de Precificação

### 1. Precificação por Custo
```
PREÇO = Custo total + Margem desejada

Exemplo:
Curso custa R$500 para produzir
Margem desejada: 80%
Preço: R$500 ÷ (1 - 0,80) = R$2.500
```

### 2. Precificação por Valor
```
PREÇO = Valor da transformação × Fator de acessibilidade

Exemplo:
Curso ajuda a conseguir emprego de R$5.000/mês
Valor em 12 meses: R$60.000
Fator: 2-5% do valor gerado
Preço: R$1.200 - R$3.000
```

### 3. Precificação por Mercado
```
PREÇO = Média dos concorrentes × Posicionamento

Concorrentes cobram R$497, R$697, R$997
Média: R$730
Posicionamento premium (+20%): R$876
Posicionamento acessível (-20%): R$584
```

## Estratégias de Preço

### 1. Preço de penetração
- Começar baixo para ganhar mercado
- Aumentar gradualmente
- Bom para novos produtos

### 2. Preço premium
- Posicionar como o melhor
- Justificar com diferenciação
- Atrair público qualificado

### 3. Preço de pacote
- Combinar produtos
- Oferecer desconto no conjunto
- Aumentar ticket médio

### 4. Preço por assinatura
- Receita recorrente
- Menor barreira de entrada
- Relacionamento longo

### 5. Preço dinâmico
- Lançamento com desconto
- Carrinho aberto/fechado
- Escassez real

## Descontos Estratégicos

### Quando dar desconto:

| Situação | Desconto sugerido |
|----------|-------------------|
| Early bird (lançamento) | 20-40% |
| Black Friday | 30-50% |
| Renovação de assinatura | 10-20% |
| Indicação | 10-15% cada |
| Pagamento à vista | 5-10% |

### Quando NÃO dar desconto:

- ❌ Sem motivo aparente
- ❌ Para todos (perde valor)
- ❌ Frequentemente (vira padrão)
- ❌ Maior que a margem

## Calculando seu Preço Ideal

### Fórmula completa:

```
PREÇO MÍNIMO = (Custos fixos + Custos variáveis + Lucro desejado) ÷ Vendas esperadas

PREÇO IDEAL = Preço mínimo × Fator de valor percebido
```

### Exemplo prático:

```
Custos fixos mensais: R$2.000
Custos por venda: R$50
Lucro desejado: R$5.000
Vendas esperadas: 50

Preço mínimo = (2.000 + (50×50) + 5.000) ÷ 50
Preço mínimo = 9.500 ÷ 50 = R$190

Com fator de valor 2.5x:
Preço ideal = R$190 × 2.5 = R$475 ≈ R$497
```

## Teste de Preço

### Como descobrir o preço ideal:

1. **Pesquisa com audiência**
   - "Quanto você pagaria por X?"
   - "Qual valor seria caro demais?"

2. **Teste A/B**
   - Duas landing pages
   - Preços diferentes
   - Medir conversão

3. **Escada de valor**
   - Começar com produto barato
   - Observar conversão para upsell
   - Ajustar preços
