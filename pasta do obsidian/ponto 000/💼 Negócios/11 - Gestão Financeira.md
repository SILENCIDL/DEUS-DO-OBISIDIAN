## Separação de Contas

### Estrutura recomendada:

```
CONTA PJ (Receitas)
├── CONTA IMPOSTOS (20-30%)
├── CONTA INVESTIMENTO (20-30%)
├── CONTA RESERVA (6 meses de custo)
└── CONTA PESSOAL (Pró-labore)
```

### Regra dos percentuais:

| Destino | % da receita |
|---------|--------------|
| Impostos | 15-30% |
| Reinvestimento | 20-30% |
| Reserva | 10-20% |
| Pró-labore | 30-50% |

## Indicadores Financeiros

### KPIs essenciais:

| Indicador | Fórmula | Meta |
|-----------|---------|------|
| Receita Bruta | Soma das vendas | Crescer mês a mês |
| Receita Líquida | Bruta - Taxas - Impostos | 70-80% da bruta |
| Margem de Lucro | Lucro ÷ Receita | 40-60% |
| CAC | Custo de aquisição ÷ Clientes | Menor possível |
| LTV | Valor médio × Tempo de cliente | Maior que CAC × 3 |
| MRR | Receita recorrente mensal | Previsibilidade |

## Impostos para Criadores

### MEI:
- **DAS mensal:** ~R$70-75 (2026)
- **Limite:** R$81.000/ano
- **Vencimento:** Dia 20 de cada mês

### Simples Nacional (ME):

| Faixa de faturamento | Alíquota |
|----------------------|----------|
| Até R$180k | 6% |
| R$180k - R$360k | 11,2% |
| R$360k - R$720k | 13,5% |
| R$720k - R$1.8M | 16% |
| R$1.8M - R$3.6M | 19% |
| R$3.6M - R$4.8M | 19,5% |

### Recebimentos do exterior:

1. **Conta internacional** (Payoneer, Wise)
2. **Converter para BRL**
3. **Declarar como receita**
4. **Pagar imposto normal**

⚠️ **Consulte um contador** para situações específicas!

## Reserva de Emergência

### Cálculo:

```
RESERVA = Custo mensal total × 6 meses

Exemplo:
Custos fixos: R$3.000
Custos variáveis: R$1.500
Pró-labore: R$5.000
Total: R$9.500

Reserva ideal: R$57.000
```

### Onde guardar:

| Opção | Liquidez | Rendimento |

| CDB Liquidez diária | D+0 | 100-110% CDI |
