## 📐 Gramática Básica

### Pronomes Pessoais

| Russo | Transliteração | Português |
|-------|----------------|-----------|
| я | ya | eu |
| ты | ty | você (informal) |
| он | on | ele |
| она | ona | ela |
| оно | ono | isso |
| мы | my | nós |
| вы | vy | vocês / você (formal) |
| они | oni | eles |

### Verbo "Ser" (БЫТЬ)

| Pessoa | Presente | Exemplo |
|--------|----------|---------|
| я | есть (yest') | Я студент. |
| ты | есть | Ты студент. |
| он/она/оно | есть | Он студент. |
| мы | есть | Мы студенты. |
| вы | есть | Вы студенты. |
| они | есть | Они студенты. |

> ⚠️ **Nota:** O verbo "ser" no presente geralmente é omitido em russo!

## 📅 Plano de Estudos — 24 Meses

### Fase 1: Alfabeto e Sons (Meses 1-2)
- [ ] Aprender alfabeto cirílico
- [ ] Dominar leitura básica
- [ ] 100 palavras essenciais
- [ ] Saudações e apresentação

### Fase 2: Fundamentos (Meses 3-6)
- [ ] 500 palavras
- [ ] Casos gramaticais básicos
- [ ] Verbos de movimento
- [ ] Frases simples

### Fase 3: Intermediário (Meses 7-12)
- [ ] 1.500 palavras
- [ ] Todos os casos
- [ ] Aspecto verbal
- [ ] Conversação básica

### Fase 4: Avançado (Meses 13-18)
- [ ] 3.000 palavras
- [ ] Gramática completa
- [ ] Séries com legendas
- [ ] Conversação fluente

### Fase 5: Fluência (Meses 19-24)
- [ ] 5.000+ palavras
- [ ] Sotaque natural
- [ ] Séries sem legenda
- [ ] Pronto para intercâmbio!
